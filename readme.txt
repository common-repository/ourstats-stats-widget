create a plugin for the ourstats.de counter service

== Description ==

ourSTATS.de is a free web Service to find out information about your visitors such as eg Screen resolution, browser, language and origin, page views by hour survey, as well as what keywords your visitors use to go to your homepage.

Create your different ourSTATS.de statistics for each section of your website. Enter your ourSTATS.de supplied ID, you can get without registering, visiting ourstats.de. To insert a gallery in your posts or pages, type [ourstats_stats X/], where X is the code of the stats, not the ourSTATS.de ID. You can use Widget with this plugin.

For create and configure the stats press in your Wordpress admin: ourSTATS plugin

<a href="http://www.pluginswp.com">Plugin web</a>		

== Installation ==

1. Upload `ourstats-stats-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure and create new video gallery in your Wordpress Admin: pera Video
4. Place [`ourstats_stats` ID /] in your Posts, where ID is the stats ID, not the ourSTATS.de ID.

5. Admin params:

	- title
	
	- ourSTATS.de ID

	- hide if admin
	
	- colors
